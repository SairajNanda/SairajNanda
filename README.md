## About

I'm a software engineer who builds intelligent software products - systems where thoughtful engineering and applied AI meet a real user problem. My work spans the full stack: designing data models and APIs on the backend, shipping polished interfaces on the front, and wiring in language models, retrieval, and machine learning where they make the product genuinely better rather than just novel.

I care about the parts that don't show up in a demo - authentication that holds, schemas that scale, real-time flows that stay consistent under load, and automation that quietly removes hours of manual work. I treat every project as a product decision first and an engineering problem second, then build the clean systems that make the decision hold up.

## Featured Projects

### Resume Building & Career Ecosystem
An AI-powered SaaS platform that generates dynamic resumes from *verified* achievements — pairing LLM-written summaries with an achievement-verification layer so output stays credible. Full-stack architecture with live updates and secure auth.

`Next.js` · `TypeScript` · `Tailwind` · `FastAPI` · `SQLAlchemy` · `PostgreSQL` · `OpenAI` · `JWT`

---

### RAG Architecture
A retrieval-augmented AI application: upload PDFs, ingest and chunk documents, embed with sentence transformers, and serve grounded answers over FAISS semantic search — with Redis caching and edge inference on Cloudflare Workers AI.

`Flask` · `FAISS` · `Sentence Transformers` · `Redis` · `Cloudflare Workers AI` · `PostgreSQL`

---

### Intervue - Live Polling System
A real-time classroom platform where teachers run live polls and students respond over WebSockets. Synchronized dashboards, in-session chat, and live poll analytics — built around consistent real-time state.

`React` · `Redux` · `Express` · `Socket.io` · `Node.js`

---

### Recipe Rating Prediction
A complete NLP + ML pipeline predicting ratings from review text — TF-IDF vectorization, sentiment features, ensemble models, hyperparameter search, and rigorous error analysis.

`Scikit-learn` · `XGBoost` · `Random Forest` · `TF-IDF` · `GridSearchCV` · `NLP`

---

### Certificate Distributor
A Python automation tool for bulk certificate generation and distribution — CSV-driven batch processing that turns a manual, hours-long task into a single run.

`Python` · `Automation` · `Batch Processing`
